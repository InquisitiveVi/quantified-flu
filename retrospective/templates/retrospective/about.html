About

Who's behind this?

This current test setup is so far being run and developed by Mad Price Ball & Bastian Greshake Tzovaras.

The idea for this project came out of some posts on the Quantified Self forum, as well as out of the discussions of the Open Humans community calls. Based on these Bastian created some preliminary data visualizations of his own data, which formed the basis for this web application that Mad & Bastian developed to scale things up.

What's the goal?

Currently the goal is to just allow you to self-visualize your own physiological signals, but once we've decided on a framework we can actually use the aggregate data to see if there's some clearer signal of the physiological signals that can be used regardless.

How can I contribute?

There's multiple ways to help out and we're looking for people in all areas. Some easy ways to get involved are:

  • Do you have retrospective data and know when you've been sick? Adding your own events is the easist way to help
  • Spread the word! Let others know that this exists!
  • Join our Slack! Chat with others interested in the Quantified Flu
  • Improve this website! There are open issues like improving the data visualizations, support for more wearables etc. Look at our issues on GitHub
  • Analyze data! Once we start aggregating data we'll need data scientists willing to help us go through the data!
